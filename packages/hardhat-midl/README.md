# Hardhat MIDL Package

This package contains smart contracts and deployment scripts for the MIDL Hackathon.

## Features

- Perpetual futures contracts
- Margin, liquidation, funding logic
- Chainlink price feeds
- Hardhat deploy scripts

## Development

```sh
pnpm install
pnpm hardhat test
pnpm hardhat deploy
```

## Build

```sh
pnpm build
```

## Configuration

- See `hardhat.config.ts` and `tsconfig.json` for project settings.
- Uses workspace dependencies from root monorepo.
